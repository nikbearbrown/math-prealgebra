## Source-level notes

- **No contradictions** between source files.
- **Pedagogical compression.** Source treats each method (graphing, substitution, elimination) as a separate section with worked examples for each. Rewrite condenses graphing to a brief treatment in Concept 1 and elevates substitution/elimination to Concept 2 with explicit selection guidance. Same content; tighter narrative.
- **Cyclist meeting problem (§1).** Original construction in the rewrite, but standard motion-system template; numbers chosen to give clean fractional answer ($t = 5/3$ hours) so that students see the method work even when the time is non-integer.
- **Candle/mug synthesis problem (§5).** Original construction. The system was deliberately constructed inconsistent so that the rewrite could demonstrate that an inconsistent system is *operationally informative*, not a math failure.
- **Boat-with-current problem.** Used as the §4 worked example, drawn from the standard motion-system template; OpenStax m82497 has similar examples.
